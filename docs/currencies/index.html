Module multicurrency.currencies

Currencies.

The Currencies module provides the implementation of the several existing currencies as extensions of Currency. It supports several different currencies.

The currencies supported by this module were created with information (alphabetic code, numeric code, and minor unit size) from ISO-4217.

Note

Cryptocurrencies are not represented in the ISO-4217. Information on cryptocurrencies was collected on other sources.
